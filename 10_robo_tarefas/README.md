# 🤖 Projeto 10 - Robô de Tarefas

## 📖 Sobre o projeto

Este projeto foi desenvolvido com o objetivo de aprender os conceitos fundamentais de automação de tarefas utilizando Python.

O robô lê informações de uma planilha Excel e executa automaticamente uma sequência de ações no computador utilizando a biblioteca **PyAutoGUI**.

Neste exemplo, o programa abre o Bloco de Notas, lê os produtos cadastrados na planilha e escreve todas as informações automaticamente.

---

## 🚀 Tecnologias utilizadas

* Python
* PyAutoGUI
* OpenPyXL
* Subprocess
