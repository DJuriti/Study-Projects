# 📦 Sistema de Estoque com Excel e Relatórios em PDF

## 📖 Sobre o projeto

Este projeto foi desenvolvido em Python com o objetivo de praticar manipulação de planilhas Excel e geração automática de documentos PDF.

O sistema permite gerenciar um estoque de produtos, armazenando os dados em uma planilha Excel e gerando um relatório em PDF contendo todas as informações cadastradas.

---

## 🚀 Funcionalidades

### Sistema de Estoque

- Cadastrar produtos
- Consultar produtos
- Atualizar quantidade
- Atualizar preço
- Listar produtos
- Remover produtos
- Salvar automaticamente os dados em uma planilha Excel

### Gerador de Relatórios

- Ler automaticamente os dados da planilha
- Gerar um relatório em PDF
- Organizar os produtos para impressão
