# Projeto 03 - Previsão do Tempo

Aplicação desenvolvida em Python que consulta informações meteorológicas utilizando a OpenWeather API.

## Funcionalidades

- Consulta por cidade
- Temperatura atual
- Temperatura máxima
- Temperatura mínima
- Sensação térmica
- Umidade
- Velocidade do vento
- Condição do tempo

## Tecnologias

- Python
- Requests
- Python-dotenv
- OpenWeather API

## Como executar

1. Instale as dependências:

```bash
pip install requests python-dotenv
```

2. Crie um arquivo `.env`

```
API_KEY=SUA_API_KEY
```

3. Execute:

```bash
python previsao_tempo.py
```
