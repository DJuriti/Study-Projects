# Dashboard de Dados

Projeto de análise de vendas utilizando Python.

## Tecnologias

- Python
- Pandas
- Matplotlib
- Streamlit
- Excel

## Funcionalidades

- Leitura de dados Excel
- Cálculo de faturamento
- Indicadores de vendas
- Gráficos de análise

## Como executar

pip install -r requirements.txt

streamlit run src/app.py
