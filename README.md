# Projeto 02 - Cotação de Moedas

Projeto desenvolvido em Python utilizando a AwesomeAPI.

## Funcionalidades

- Consulta de moedas
- Compra
- Venda
- Cotação máxima
- Cotação mínima
- Variação percentual
- Data da atualização

## Tecnologias

- Python
- Requests
- Python-dotenv
- AwesomeAPI

## Como executar

1. Clone o repositório
2. Instale as dependências

```bash
pip install requests python-dotenv
```

3. Crie um arquivo `.env`

```
API_KEY=SUA_API_KEY
```

4. Execute

```bash
python cotacao_moedas.py
```
